# Greenhouse Operations Console (greenhouse-ops-rootfix-0508c) ## Project Overview Maintenance planning application for greenhouse operations. English language. ## Screen List 1. **Dashboard**: Primary overview with summary cards (Total Tasks, Equipment Health, Pending Maintenance) and main actions. 2. **Task Board**: Kanban-style board with work items grouped by status (Todo, In Progress, Done). 3. **Equipment Status**: List/Grid view of equipment with health indicators (sensors, power, irrigation). 4. **Maintenance Log**: Chronological table of activities with search and filters. 5. **Filtered Overview**: Search-focused view showing filtered operational results. 6. **Storage Error State**: Error screen for localStorage persistence failures with Retry/Reset buttons. 7. **Profile Panel**: Modal/Panel for account details and notification toggles. 8. **Settings**: Application preferences and storage management. 9. **Empty State**: Guidance screen for first-time users or no-results found. ## Visual Direction - Dark, modern theme. - High contrast for readability. - Professional tool aesthetic.
